# Как поднять:

## 1. Запуск контейнера с PostgreSQL, Nats, Clickhouse, Redis:
```
make compose
```
## 2. Миграция таблиц в поднятой Postgres:
```
make upmigrate
```
## 3. Запуск сервера:
```
make run
```
## 4. Остановка контейнеров:
```
make stop
```
